# Arduino Car Part 2

Projeto Carrinho Seguidor de Linha Arduino para faculdade UNIP

2 opções de arquivo; formato (.cpp) é C++ e C formato (.ino) é padrão arduino

Atualização 27/11/2025
Projeto apresentado segunda etapa, apresentado 24/11 e 25.
Segundo fase incluem: 3 sensores infravermelhos, 1 ultrassonico, 1 leitor RFID, 1 Display LCD, Ponte H e Arduíno Uno R3, programado em C para Arduíno.
